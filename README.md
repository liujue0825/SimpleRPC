# lj-rpc-framework

## 项目简介

- 一款基于 Netty + Zookeeper + SpringBoot 实现的自定义 RPC 框架。



## 基本内容

- [x] 基于Netty实现网络传输，并实现了服务的优雅启动和优雅下线；
- [x] 自定义了消息协议和编解码器；

- [x] 使用 Zookeeper/Nacos 作为注册中心；
- [x] 实现了多种序列化算法，包括JDK，JSON，Hessian，Protobuf，Kryo等；
- [x] 实现了两种常用的动态代理方式，包括JDK 和 CGlib；
- [x] 实现了四种常用的负载均衡策略，包括轮询，随机选择，LRU和一致性哈希；
- [x] 优化网络传输过程，包括解决黏包半包、添加心跳机制、复用channel以维持长连接；
- [x] 集成 Spring，自定义注解来辅助实现 RPC 组件扫描、服务注册、服务消费等；
- [x] 集成 SpringBoot，完成自动配置；
- [x] 实现自定义的SPI机制以适应进一步扩展和偏好使用。
- [ ] 添加服务监控中心；
- [ ] 添加服务容灾功能；
